# Smart Grid Automation using AI & IoT

This project demonstrates an intelligent smart grid system using IoT data
and AI models to automate load management and fault detection.

## Features
- Real-time IoT data simulation
- AI-based overload detection
- Automated grid control logic
- Interactive dashboard

## Tech Stack
- Python
- AI / Machine Learning
- IoT Sensors (Simulated)
- Streamlit
